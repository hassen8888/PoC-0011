# PoC-0011_Request（方向を見ないEntry & 継続動意ExitのPoC要求仕様）

## 1. PoCの目的
方向を一切見ない Entry 条件でポジションを持ち、first-touch（±0.03）以降の「継続動意」を用いて Exit（利確・手仕舞い）条件の特徴・ルール候補を見出す。

- Entry：方向（UP/DOWN）を見ない
- Exit：first-touch後の「継続動意」で利確幅を調整
- 成果物：継続動意に基づく Exit ロジックの候補と統計的裏付け

---

## 2. 対象シンボル・期間・データ粒度
**シンボル:** EURUSD（PoC-0010 と同一前提）  
**期間:** PoC-0010 と同一期間  
**データ粒度:** Tickデータ（内部処理）、10秒足・1分足・5分足を使用

---

## 3. Entryロジック（方向を見ないEntry条件）

### 3.1 基本方針
- Entry時点では方向を一切見ない
- Entry条件は「動く準備が整っている状態（動意が高い状態）」のみ
- Buy/Sell方向は first-touch で事後的に決まる（分析用ラベル）

### 3.2 Entry判定に使う情報
**10秒足の動意（中心）**
- 直近10秒の値幅（High-Low）
- 10秒足MACD（main, signal, hist）
- 10秒足STOCH（%K, %D）
- 10秒足のボラティリティ（標準偏差など）

**スプレッド条件**
- Spread ≤ 3

**時間帯条件**
- 欧州〜NY前半など「動きやすい時間帯」
- PoC-0010で動かないと判定された時間帯（22〜23時、6〜8時）は除外候補

**直近レンジ判定**
- 直近60秒の値幅が極端に小さい場合は除外
- 小さな往復のみの状態（レンジ）は除外

### 3.3 Entryトリガー（候補）
以下のような条件を満たしたTickで Entry 発生としてログ記録：
- 直近10秒の値幅 ≥ X
- 10秒足MACDの絶対値 ≥ A
- 10秒足STOCHが極端（≤20 または ≥80）
- Spread ≤ 3
- 時間帯が「動意あり」ゾーン内

※ この段階では方向は未定義。方向ラベルは first-touch で決まる。

---

## 4. first-touch定義と方向ラベル

### 4.1 first-touchの定義
Entry価格を基準に：
- +0.03 到達 → 上方向 first-touch
- -0.03 到達 → 下方向 first-touch
- 規定時間（例: 10分）以内にどちらにも触れない → NoTouch

### 4.2 方向ラベル
- +0.03 → direction = UP
- -0.03 → direction = DOWN
- NoTouch → direction = NONE

---

## 5. 継続動意（Exit候補）の定義

### 5.1 継続動意の目的
first-touch で方向が確定した後、「その方向にどれだけ伸びるか」を評価し、Exit（利確・手仕舞い）のルール候補を見出す。

### 5.2 継続動意に使う時間足と役割
**10秒足**
- first-touch前後の短期動意
- first-touchまでの速度
- first-touch後の直近値幅

**1分足**
- 方向バイアス（伸びやすさ）
- first-touch後の伸び幅に影響

**5分足**
- 背景トレンド（伸びの上限・持続性）

### 5.3 継続動意の指標候補
- first-touch後の最大伸び幅（同方向）
- first-touch後の最大逆行幅（逆方向）
- first-touch後の一定時間内（1分/3分/5分）の同方向累積値幅・逆方向累積値幅
- 10秒足MACD/STOCHの変化量（dMACD/dSTOCH）
- 1分足・5分足の方向整合性（first-touch方向と一致/逆/中立）

---

## 6. ログ仕様（PoC-0011用）

### 6.1 Entryログ
- 時刻
- Bid/Ask/Spread
- 10秒足の状態（値幅、MACD、STOCH）
- 1分足・5分足の簡易状態（環境認識）
- 時間帯ラベル
- Entry ID

### 6.2 first-touchログ
- Entry ID
- first-touch方向（UP/DOWN/NONE）
- 到達時間（秒）
- 到達までの最大逆行幅
- first-touch時点の10秒足/1分足/5分足の状態

### 6.3 継続動意ログ
- Entry ID
- first-touch後の観測ウィンドウ（1分/3分/5分）ごとに：
  - 同方向最大伸び幅
  - 逆方向最大逆行幅
  - 同方向/逆方向の累積値幅
  - 10秒足MACD/STOCHの推移要約
  - 1分足・5分足の方向整合性
